A custom linux command "internsctl".

## Overview

Created a custom linux command "internsctl"

## Features

Can use various commands like
-- version
-- help
cpu getinfo
memory getinfo
user create
user list
file getinfo


## Prerequisites

Before you begin, ensure you have met the following requirements:

- Linux installed (or git bash, wsl)
